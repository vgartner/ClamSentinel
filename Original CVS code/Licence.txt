Clam Sentinel est un scanner en temps réel pour ClamWin.
Pour Windows 98/98SE/ME/2000/XP/Vista/Windows 7, Windows 8.

Clam Sentinel nécessite l'installation de ClamWin <http://www.clamwin.com/>.
