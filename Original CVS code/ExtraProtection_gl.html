Conseguir  Protección Adicional Con Clam Sentinel

Empecemos polo básico. Fixe a frecuencia de actualización da base de datos ClamWin a "hourly" (Actualizar cada hora). Use a última versión de ClamWin e Clam Sentinel. Execute un análise de memoria diario en ClamWin , cartafol system 32  (e cartafol WOW en Windows 8), e o cartafol users/documentos. Fixe Clam Sentinel para autoactivarse en cada inicio de Windows, detectar e Supervisar novas unidades, ficheiros de cuarentena infectados, e configure só para "Supervisar para detectar ficheiros sospeitosos". Configure o súa computadora para que de maneira automática se instalen as actualizacións os parches de Windows. Deixe o seu firewall acendido. O firewall Microsoft ven con Windows XP/Vista/Windows 7 protexe sobre contactos inautorizados con destino a  Internet. É dificil configuralo para unha protección con destino exterior, pero é simple, gratuíto, e “non molesta.” Hai unha versión gratuíta de software de firewall para Windows XP e computadoras máis novas  o cal é o Zone Alarm e debe ser para sistemas antigos o Zone Alarm version 6.1.744.001 que traballa en Windows 98.

Configure o seu navegador Web para Usar Filtros que teñan--como bloqueador publicidade, protección de rastrexo, filtro de phishing, protección de descargas amañadas, e protección de páxinas Web maliciosas. Mantendo o seu navegador actualizado, é mais importante que o navegador que use. Algunhas barras de ferramentas gratuítas para o navegador e engadidos proveñen protección Web adicional, pero as barras de ferramentas das compañías coñecidas como Google, Microsoft, Yahoo, ou as das compañías de antivirus. Alóxese dos sitios que ofrecen pornografía, software ilegal ou craqueado, e descargas torrent xa que poden conter virus. NUNCA dea información persoal nos sitios de redes sociais.

ClamWin/Clam Sentinel proven  unha boa protección contra a descargas de virus, pero os novos virus poden a veces evadir os programas, entón EU recomendo unha protección adicional. Usuarios de Windows XP e sistemas operativos mais novos poden facer unha boa suite de seguridade con estes programas gratuítos: WinPatrol, Malwarebytes Antimalware, Malwarebytes Antirootkit, e Malwarebytes Anti-Exploit. WinPatrol da compañía BillP Studios é un o programa para supervisar intrusións de maneira gratuíta—use por defecto, e supervisará cambios na súa computadora. Execute a parella de análises diarios Malwarebytes Antimalware despois de actualizalo. Faga unha análise co Malwarebytes Antirootkit algunhas veces cada semana para remover algún virus oculto. Malwarebytes Anti-Exploit protexe ós usuarios contra exploits outra vez durante un rato cando navegue pola Web. Se para todos isto é moito para Vd para usar, pode usar usar Malwarebytes Antimalware con ClamWin / Clam Sentinel. Para computadoras máis vellas que o Windows XP, pódolle suxerir que sempre use a última versión de ClamWin, e  configuéreo para que as actualización da base de virus se faga a cara hora  ClamWin para actualizacións automáticas, e faga un análise diario con ClamWin no "modo a prueba de fallos" ou “Modo Seguro” de Windows. Finalmente, é recomendable que use un CD/USB de rescate que están dispoñibles dende algunhas compañías de Antivirus  (como as F-Secure, Dr. Web, e Kaspersky). Eu prefiro o Microsoft's Windows Defender Offline Gratuíto que  facer copias de rescate en CD or USB (Eu fíxeno no USB). Vd debe executar o CD/USB de rescate CD/USBs unha conexión DSL por cable para actualizar propiamente as sinaturas de virus antes de analizar a computadora. Lea a documentación do CD/USB de recate antes de proceder. Execute un análise de busca co  CD/USB rescate.

Finalmente, gardar na mente que unha navegación e sentido común preveñen a mellor protección de todo.

Por favor contacte con Andrea Russo, o desenrolador de Clam Sentinel, pola páxina de  Clam Sentinel se vostede ten algún comentario acerca desde artigo ou suxestións para mellorar Clam Sentinel. Os programadores en Delphi poden axudar a contribuír ó proxecto de Clam Sentinel  contactando con Andrea a través da páxina Web de Clam Sentinel.

Clam Sentinel é un proxecto separado de ClamWin, pero para que este funcione, ClamWin debe ser instalado primeiro.
