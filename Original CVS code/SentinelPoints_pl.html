Clam Sentinel

Darmowy oraz w pełni funkcjonalny skaner w czasie rzeczywistym dla programu antywirusowego ClamWin
  • Dodaje funkcję skanera w czasie rzeczywistym do programu ClamWin
  • Zintegrowany system wykrywania włamań (Intrusion detection)
  • Aktywna ochrona heurystyczna
  • Ochrona dysków wymiennych oraz USB
  • Używa folderu kwarantanny programu ClamWin
  • Tworzy dzienniki ze skanowania rzeczywistego, skanowania dysków, pamięci i wiadomości
  • Domyślna konfiguracja odpowiednia dla większości użytkowników
  • Łatwa konfiguracja poprzez ikonę w zasobniku systemowym (System Tray)
  • Obsługa Windows 98 oraz nowszych komputerów
  • Program po polsku, angielsku, włosku, francusku, niemiecku oraz japońsku
  • Wsparcie wielu użytkowników
