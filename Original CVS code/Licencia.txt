Clam Sentinel es un escáner en tiempo real para ClamWin.
Para Windows 98/98SE/ME/2000/XP/Vista/Windows 7 y Windows 8.

Clam Sentinel requiere la instalación de ClamWin <http://es.clamwin.com/>.
